# Code of Conduct

NexusChats soll ein freundliches, respektvolles und sachliches Open-Source-Projekt sein.

Dieser Code of Conduct gilt fuer Issues, Pull Requests, Reviews, Discussions, Releases und andere Projektkanaele.

## Erwartetes Verhalten

- respektvoll kommunizieren
- konstruktiv kritisieren
- unterschiedliche Erfahrungsstaende beruecksichtigen
- technische Entscheidungen anhand von Fakten und Projektzielen diskutieren
- Datenschutz und Nutzervertrauen ernst nehmen
- gute Absichten annehmen, solange nichts dagegen spricht
- Fehler offen korrigieren

## Nicht akzeptiertes Verhalten

- Beleidigungen, Belaestigung oder persoenliche Angriffe
- diskriminierende Sprache
- absichtliches Stoeren von Diskussionen
- Veroeffentlichung privater Informationen anderer Personen
- Druck, private Daten oder Chat-Inhalte offenzulegen
- Beitraege, die Tracking, Werbung oder heimliche Datenuebertragung einfuehren
- wiederholtes Ignorieren von Maintainer-Entscheidungen nach erklaerter Begruendung

## Datenschutz als Community-Regel

NexusChats ist Privacy First. Diskussionen, Issues und Tests sollen keine echten privaten Chat-Inhalte enthalten. Verwende kuenstliche Beispiele oder anonymisierte Daten.

## Durchsetzung

Maintainer koennen:

- auf problematisches Verhalten hinweisen
- Diskussionen moderieren
- Kommentare ausblenden oder entfernen
- Issues oder Pull Requests schliessen
- Personen zeitweise oder dauerhaft von Projektkanaelen ausschliessen

Massnahmen sollen verhaeltnismaessig, nachvollziehbar und auf den Schutz der Community ausgerichtet sein.

## Meldung

Verhaltensprobleme koennen ueber ein Issue gemeldet werden, sofern keine privaten Informationen betroffen sind. Wenn private Informationen oder Sicherheitsdetails betroffen sind, verwende die Hinweise in [SECURITY.md](SECURITY.md).
