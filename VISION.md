# Vision

NexusChats soll die beste kostenlose Open-Source-Erweiterung für die Verwaltung von ChatGPT-Unterhaltungen werden.

Das Projekt verbessert ChatGPT nicht durch mehr Ablenkung, sondern durch Ordnung, Geschwindigkeit und Vertrauen. Jede Funktion soll Nutzern helfen, lange Chat-Verläufe besser zu strukturieren, wiederzufinden und sicher zu verwalten.

## Grundprinzipien

- Local First
- Privacy First
- Open Source
- kein Tracking
- keine Werbung
- keine Cloud
- keine versteckten Premium-Funktionen
- keine Serverkommunikation für Kernfunktionen
- keine Speicherung unnötiger Daten

## Produktversprechen

NexusChats bleibt ein Werkzeug für Produktivität und Organisation. Es soll die vorhandene ChatGPT-Oberfläche ergänzen, nicht ersetzen.

Das Projekt priorisiert:

- schnelle Orientierung in langen Chatlisten
- lokale Auswahl-, Markierungs- und Organisationsfunktionen
- klare, ruhige UI-Ergänzungen
- nachvollziehbare Aktionen
- sichere Defaults
- wartbare Architektur

## Datenschutzversprechen

NexusChats darf nur die Daten verarbeiten, die für lokale UI-Funktionen notwendig sind.

Das bedeutet:

- keine Analytics
- keine Telemetrie
- keine Remote-Konfiguration
- keine externen KI- oder Daten-APIs
- keine Übertragung von Chat-Inhalten
- keine versteckten Netzwerkaufrufe

Wenn eine zukünftige Funktion neue lokale Speicherung benötigt, muss sie dokumentiert und im Pull Request begründet werden.

## Open-Source-Kultur

NexusChats soll langfristig von einer Community gepflegt werden können. Entscheidungen sollen transparent, nachvollziehbar und freundlich diskutiert werden.

Gute Projektarbeit bedeutet hier:

- kleine, reviewbare Pull Requests
- klare Architekturgrenzen
- Tests für riskante Logik
- Datenschutzprüfung bei jeder Funktion
- respektvoller Umgang mit Nutzern und Beitragenden

## Nicht-Ziele

- keine Werbeplattform
- keine SaaS-Synchronisierung
- keine Account- oder Profilbildung
- keine Umgehung von ChatGPT-Sicherheits- oder Nutzungsgrenzen
- keine manipulative UI
- keine Abhängigkeit von zufälligen ChatGPT-CSS-Klassen

## Langfristiges Ziel

NexusChats soll sich anfühlen wie ein verlässlicher lokaler Werkzeugkasten für ChatGPT: offen, transparent, schnell, respektvoll und dauerhaft nutzbar.
