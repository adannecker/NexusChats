# Vision

NexusChats soll die beste kostenlose Open-Source-Erweiterung fuer die Verwaltung von ChatGPT-Unterhaltungen werden.

Das Projekt verbessert ChatGPT nicht durch mehr Ablenkung, sondern durch Ordnung, Geschwindigkeit und Vertrauen. Jede Funktion soll Nutzern helfen, lange Chat-Verlaeufe besser zu strukturieren, wiederzufinden und sicher zu verwalten.

## Grundprinzipien

- Local First
- Privacy First
- Open Source
- kein Tracking
- keine Werbung
- keine Cloud
- keine versteckten Premium-Funktionen
- keine Serverkommunikation fuer Kernfunktionen
- keine Speicherung unnoetiger Daten

## Produktversprechen

NexusChats bleibt ein Werkzeug fuer Produktivitaet und Organisation. Es soll die vorhandene ChatGPT-Oberflaeche ergaenzen, nicht ersetzen.

Das Projekt priorisiert:

- schnelle Orientierung in langen Chatlisten
- lokale Auswahl-, Markierungs- und Organisationsfunktionen
- klare, ruhige UI-Ergaenzungen
- nachvollziehbare Aktionen
- sichere Defaults
- wartbare Architektur

## Datenschutzversprechen

NexusChats darf nur die Daten verarbeiten, die fuer lokale UI-Funktionen notwendig sind.

Das bedeutet:

- keine Analytics
- keine Telemetrie
- keine Remote-Konfiguration
- keine externen KI- oder Daten-APIs
- keine Uebertragung von Chat-Inhalten
- keine versteckten Netzwerkaufrufe

Wenn eine zukuenftige Funktion neue lokale Speicherung benoetigt, muss sie dokumentiert und im Pull Request begruendet werden.

## Open-Source-Kultur

NexusChats soll langfristig von einer Community gepflegt werden koennen. Entscheidungen sollen transparent, nachvollziehbar und freundlich diskutiert werden.

Gute Projektarbeit bedeutet hier:

- kleine, reviewbare Pull Requests
- klare Architekturgrenzen
- Tests fuer riskante Logik
- Datenschutzpruefung bei jeder Funktion
- respektvoller Umgang mit Nutzern und Beitragenden

## Nicht-Ziele

- keine Werbeplattform
- keine SaaS-Synchronisierung
- keine Account- oder Profilbildung
- keine Umgehung von ChatGPT-Sicherheits- oder Nutzungsgrenzen
- keine manipulative UI
- keine Abhaengigkeit von zufaelligen ChatGPT-CSS-Klassen

## Langfristiges Ziel

NexusChats soll sich anfuehlen wie ein verlaesslicher lokaler Werkzeugkasten fuer ChatGPT: offen, transparent, schnell, respektvoll und dauerhaft nutzbar.
